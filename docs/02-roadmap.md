# Success Criteria

The MVP is successful if:

- Users complete tasks daily
- Habits maintain >60% completion
- DAU grows consistently
- Crash rate <1%
- Fast load times (<2s)
- Positive user feedback

---

# Risks

- Feature creep
- Over-engineering
- AI costs
- Notification fatigue
- Data privacy

Mitigation:

- Focus on MVP first.
- Validate features with users.
- Monitor AI usage.
- Keep privacy transparent.
